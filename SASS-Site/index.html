Situation-Aware Stop Signal (S.A.S.S.)

A new apprach to controlling traffic.
We wanted to design a system that would help schedule traffic and protect drivers using modern sensor technologies.
Learn More

Prevent, Protect, & Schedule

We wanted to create a system that would prevent and protect against accidents at intersections.

Self-Sustaining

Finally we wanted this system to be self-sustainable using solar power.

Features

Solar Powered

We use a custom built monocrystalline solar panel to help charge our battery backup and power our device. For us this was an important feature beacause each year intersections with lights cost hundreds of thousands of dollars in tax payer money. Removing this operating cost allows that money to be used elsewhere for more important projects and improve roadways.

RADAR Technology

We use modern mmWave radar technologies to detect and classify vehicles. This is important because it allows us to prevent accidents by classifying vehicles as safe or unsafe. By doing this we allow our system to protect drivers when unforeseen threats arrive.

LiDAR Technology

With advances in LiDAR technology in recent years we were able to develop this system with the LiDAR-Lite v3HP sensor that allowed us to view vehicles up to 40 meters away at pinpoint locations. In our design, we use the LiDAR to detect vehicles at the intersection and using that information we are able to schedule vehicles that have stopped on time in the intersection.

Modular

We designed the system to be modular allowing buyers to purchase the outer panels independently of the main control box. This aids in installation time and cost since each unit can be customized for a given intersection.

Real-Time Operation

The entire system relies on a Real-Time Operating System (RTOS) called TI-RTOS that allows the system to have deterministic runtimes, which guarantees that each component of the system operates as expected, in real time, with little to no errors.

For the Future...

There is much more to this project and it hopefully inspires others to think about traffic control at all intersections. We fully believe that these newer technolgies and the engineers of today and tomorrow can create more robust traffic control devices that help prevent and protect against accidents on our roadways.
